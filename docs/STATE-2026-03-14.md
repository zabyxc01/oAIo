### Files

```
src/
  main/

  renderer/
    index.html        — Layout: avatar (left) + chat bubbles (right) + text input
    renderer.src.js   — Three.js + VRM rendering + animation loop + lip sync integration
    voice/
      config.js       — Waifu persona config (model, voice, TTS backend, URLs)
      mic.js          — getUserMedia + MediaRecorder capture
      stt.js          — POST to faster-whisper:8003/transcribe
      llm.js          — Streaming Ollama /api/chat with sentence chunking
      tts.js          — Plug-and-play dispatcher (kokoro-rvc | indextts)
      lipsync.js      — AudioContext AnalyserNode → amplitude → VRM mouth blend shapes
      pipeline.js     — State machine: IDLE → LISTENING → PROCESSING → SPEAKING
```

### Voice Pipeline (current: IndexTTS backend)

```
F2 (start) → Mic capture (webm/opus)
F2 (stop)  → faster-whisper:8003/transcribe → user text
           → Ollama:11434/api/chat (streaming, sentence chunking)
           → For each sentence:
               → IndexTTS:8004/synthesize (ref audio + text → WAV)
               → Queue audio for sequential playback
           → Audio plays → AnalyserNode → VRM mouth blend shapes (aa, ih, ou)
           → Audio ends → IDLE
```

### Alternative Pipeline (kokoro-rvc backend)

```
Same STT + LLM flow, but TTS step:
           → oAudio:8002/speak (text + voice:af_heart + model:GOTHMOMMY.pth)
           → Kokoro-TTS:8000 → RVC:8001 → MP3
```

## Philosophy

**Gaming is the host. Compute lives in containers. Nothing crosses the boundary.**

- RADV (Mesa) is the ONLY Vulkan driver on the host
- ROCm exists ONLY inside Docker containers via /dev/kfd + /dev/dri passthrough
- /etc/environment stays clean — PATH only, no Mesa/RADV/HSA vars
- Enforcement loop pauses when no mode active — safe for gaming
- All volumes through /mnt/oaio/* — zero hardcoded personal paths in compose
